# Password Reset Guide

To reset a password, the user must verify account ownership.

Password reset steps:
1. Go to the password reset page.
2. Enter the registered username, email address, or mobile number.
3. Request a verification code.
4. Enter the code received by email or SMS.
5. Set a new password.
6. Login again using the new password.

If the user does not receive the verification code, they should check spam, confirm the entered email or mobile number, and request a new code after a few minutes.

For security reasons, support agents should not ask users to share their password or verification code.
